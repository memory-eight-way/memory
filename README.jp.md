The art of war
	base - 覚えるテキスト
	lineneumber - 行番号のみのファイル
	
使い方
    1) base のファイルの内容を行番号に関連付けて記憶
    2) linenumber  のファイルを 20190210 のような日付フォルダにコピーして回答を書き込む
    3) WinMerge などのファイル比較ソフトでしっかり覚えているか確認する
    
使い方２
    記憶する量が多くてモチベーションが下がるようなら切り出す
    1) baseのファイルを元に　特定の行番号を切り出したファイルを作成
    2) linenumberのファイルを元に　特定の行番号を切り出したファイルを作成
    3) MyFavorite20190210 のようなフォルダにコピー
    
    上記を元に記憶する
